O SISCODIS - Sistema de Controle e Distribuição é um sistema que foi desenvolvido na plataforma Web, visando a eficiência do processo de distribuição de salas. Oferece diversas opções para faciltar o processo de distribuição, entre essas opções está as funcionalidades finalizar semestre, localizar turma, bloquear sala, além de ser flexível a qualquer mudança estrutural, o que torna prática as atividades do dia-a-dia de Instituições de Ensino Superior.
